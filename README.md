## Installation

### 1. Clone the repository

```bash
git clone https://github.com/aricketts99/PARTOPT-plus.git
cd PARTOPT-plus
```

### 2. Create the conda environment

```bash
conda env create -f environment.yml
```

### 3. Activate the environment

```bash
conda activate geo_work
```

### 4. Install the package

```bash
pip install . --no-deps
```
